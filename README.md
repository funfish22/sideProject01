# Project 使用 React 框架

## Project setup

1. 安裝
    ```
    yarn
    ```
2. 啟動
    ```
    npm run serve
    ```

## 使用套件

1. [material-ui - ul 框架](https://material-ui.com/zh/)
2. [redux](https://chentsulin.github.io/redux/index.html)
3. [reduxsauce](https://github.com/jkeam/reduxsauce)
4. [swiper](https://swiperjs.com/react)
